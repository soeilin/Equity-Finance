# Variance Reconciliation (Budget vs Actual)
Built monthly Actual-vs-Budget bridge with variance-to-budget reconciliation notes and NOI-to-AFFO impact.

## Objective  
To explain earnings variance across QTD and full-year periods for a REIT-like operating platform.

## Framework  
1. Rental revenue variance  
   • Volume (occupancy, footfall proxy)  
   • Price (rent uplift, renewals)  
   • Timing (commencements/slippage)

2. Expense variance  
   • Recoveries  
   • Controllable vs non-controllable OpEx  
   • One-offs or seasonality

3. Interest & financing effects  
   • Rate variance  
   • Timing of drawdowns  
   • Refinancing impacts  

4. Flow-through to NOI, FFO, AFFO  

## Outputs  
• Variance bridge  
• +bp / –bp attribution  
• Explanatory notes suitable for investor updates or internal S&OP reporting

## Skills Demonstrated  
Variance attribution, financial diagnostics, investor-style communication.
