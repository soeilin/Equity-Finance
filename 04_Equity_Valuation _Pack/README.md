# Equity Valuation Pack (DCF + NAV + Multiples)

## Objective  
To value a REIT or real-asset platform using multiple valuation approaches and assess sensitivity to key operating assumptions.

## Valuation Methods  
1. NAV / cap-rate method  
2. DCF (simplified 10-year model)  
3. Peer multiples (P/AFFO, P/NAV, Dividend Yield)  
4. Implied yield vs sector trading range

## Key Sensitivities  
• Cap-rate ±50–100 bps  
• Discount rate adjustments  
• Rental growth scenarios  
• AFFO payout assumptions  

## Outputs  
• Valuation summary table  
• NAV premium/discount  
• Sensitivity charts  
• Equity research narrative (lite)

## Skills Demonstrated  
Valuation logic, scenario thinking, real-asset investment understanding.
